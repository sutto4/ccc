# Getting Started

Welcome to ServerMate! This guide will help you set up ServerMate in your Discord server and get started with the basic configuration.

## 📋 Prerequisites

Before you begin, make sure you have:
- A Discord server where you have Administrator permissions
- Access to the [ServerMate web dashboard](https://app.servermate.gg)

## 🚀 Step 1: Add ServerMate to Your Server

1. Visit the [ServerMate Invite Link](https://discord.com/api/oauth2/authorize?client_id=YOUR_CLIENT_ID)
2. Select the server you want to add ServerMate to
3. Click "Authorize" and complete the CAPTCHA
4. ServerMate will join your server and send you a welcome DM

## 📨 Step 2: Welcome DM

After adding ServerMate, you'll receive a direct message with:
- Getting started guide
- Links to the web dashboard
- Documentation and support resources

## 🔧 Step 3: Access the Web Dashboard

1. Go to [app.servermate.gg](https://app.servermate.gg)
2. Sign in with your Discord account
3. Select your server from the list

## ⚙️ Step 4: Initial Configuration

### Enable Basic Features

1. Navigate to your server's **Settings** page
2. Go to the **Commands & Features** tab
3. Enable the features you want to use:
   - **Moderation** - Basic moderation commands (kick, ban, warn, mute)
   - **Utilities** - Helpful utility commands
   - **Verification** - User verification system

### Set Up Permissions

1. Go to **Permissions** in your server settings
2. Configure which roles can use which commands
3. Set up role hierarchies for moderation actions

## 🎯 Step 5: Test Your Setup

Try some basic commands in your server:

```
/warn @user Breaking server rules
/kick @user Reason: Spamming
/ban @user Reason: Harassment
```
